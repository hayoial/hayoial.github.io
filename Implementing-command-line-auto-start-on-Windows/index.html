Win下实现命令行自启动

创建以下脚本，.vbs格式。

比如我在 hexo 文件夹创建这样一个脚本，命令写”hexo s”，然后给这个 .vbs 文件创建一个快捷方式，放在 shell:startup 文件夹，这样就能实现开机自启动 hexo 本地服务端了。
