小整一下博客

4月 3 2023 (4月 3 2023) digital 1 分钟 读完 (约 188 字)

把 github pages 的 cname 问题解决了，以前每次 deploy 以后自定义域名就失效，原来是必须要 CNAME 是大写才行。而如果上传了小写的，再改成大写重新部署也不行，因为 win 识别不出来大小写是否有变化。
于是先删除 cname push 一次，再重新把大写的 CNAME 放到 source 目录重新部署一次，就解决了。

又把 hexo 的 permalink 都改了一下，原来的直接就是http://blog.website.com/post-name/ 现在改成了 http://blog.website.com/post/post-name/，不光视觉上对齐了，repo 里的目录格式也整洁多了。

还把过去文章的分类也加上了。
