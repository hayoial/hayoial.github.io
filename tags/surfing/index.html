网站打不开的排查思路

3月 29 2023 (3月 29 2023) digital 1 分钟 读完 (约 155 字)
  1. 先通过这几个网站查询是否能被正常访问。
    • 海外网站：Ping Test Tool | Ping Websites, URL, Servers for Free- Site24x7 Tools
    • 国内网站：17CE.COM
    • 网站真故障了吗,还是只有你不能用？
  2. 如果网站服务器本身没有问题，通过多个dns服务器查询网站ip
  3. 如果只有一个ip(没有CDN的网站一般是这样)，检查和本机nslookup的ip是否一致，判断是否被污染。

100%能下载网页视频的方法

3月 5 2023 (3月 5 2023) digital 3 分钟 读完 (约 458 字)

仅作一个记录，软件如何安装和配置在此不表。

工具：

抓包工具 Whistle

步骤：

  1. 浏览器加载你要下载 m3u8 视频的网页。
  2. 进入 whistle 后台，左下角填”.m3u8”以过滤，在 Type 栏寻找相应的格式类型。
  3. 找到后复制 URL。
  4. 打开这个网站 M3U8 下载器，填入刚才获取的 url。选择保存路径和文件名，点击下载！(可选：另一个M3U8下载软件 N_m3u8DL-CLI，功能更强大，但基础的下载前者网站已经够用。）

Done，下载好的视频是 .ts 格式，网站已经自动将视频碎片合并成一个大文件了。

https:// *.m3u8 是个好东西，获取到它就能把视频下载到本地，也能使用本地播放器如 potplayer 直接播放。
下载下来的 m3u8 文件里如果包含主机域名，也能用该文件实现视频的下载和播放。
如果不是为了抓包别的设备，其实浏览器自带的审查元素就能过滤到.m3u8文件。

备注：
为什么明明这个网站本来就有配套的插件，还要自己费劲抓包。其实最早是在微信群里别人发的二维码里看到的一个想要下载的视频，但iOS上没有办法下载，偏偏这个链接只在手机端微信里能打开，PC端无论怎么手动更改User-Agent都进不去网页，才出此下策在手机端抓包了网页的m3u8地址。
